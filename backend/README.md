# Ex-change  
### Community-Based Skill Exchange Platform

Ex-change is a web-based platform that allows individuals to exchange skills without monetary transactions.  
Users offer skills they have and request skills they want to learn, fostering community-driven and affordable learning.

---

## Problem

Many people have valuable skills but lack access to affordable learning opportunities.  
Existing platforms focus on paid services, excluding communities that rely on mutual aid.

---

## Solution

Ex-change enables:
- Skill-for-skill exchanges (no money involved)
- Fair exchanges based on time or effort
- Community collaboration and empowerment

---

## Key Features

- User authentication  
- Skill listing (offer & request)  
- Skill exchange requests and approvals  
- In-app messaging  
- Ratings and reviews  
- Admin moderation  

---

## Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** PostgreSQL  
**Authentication:** JWT  

---

## System Users

- **User:** Lists skills and exchanges knowledge  
- **Admin:** Moderates users, skills, and disputes  

---

## Development Methodology

Agile development with iterative weekly milestones.
